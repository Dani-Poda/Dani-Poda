### 📌 Proyectos Destacados

#### 🎭 [La Simulación - Gestión de Eventos Culturales](https://github.com/Dani-Poda/MVP-La-Simulacion)
Proyecto final del bootcamp: Sistema completo de gestión de eventos culturales
- **Stack:** React + Node.js + Express + MySQL
- **Features:** Arquitectura full-stack con separación de responsabilidades, gestión de usuarios y eventos
- **Aprendizajes clave:** Estructura profesional de proyecto, buenas prácticas backend/frontend

#### 📋 [Mini-Trello - Gestión de Tareas Colaborativas](https://github.com/Dani-Poda/mini-trello)
Sistema tipo Kanban con funcionalidades avanzadas de colaboración
- **Stack:** React + Node.js + Express + MySQL + @dnd-kit
- **Features:** 
  - Drag & drop de tareas entre columnas
  - Sistema de autenticación y autorización
  - Gestión de equipos e invitaciones
  - Asignación de tareas a miembros
  - Validación con Zod schemas
- **Destacado:** Implementación completa de flujo colaborativo real

#### 📚 [Comic Book Store](https://github.com/Dani-Poda/comic-book-store)
Tienda de cómics desarrollada con Angular
- **Stack:** Angular + TypeScript
- **Features:** Routing, servicios, componentes reutilizables, gestión de estado

#### 🚗 [Car Dealership App](https://github.com/Dani-Poda/car-dealership)
Aplicación de gestión de concesionario
- **Stack:** Angular + TypeScript
- **Features:** CRUD completo, gestión de inventario, filtrado avanzado
